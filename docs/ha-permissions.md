Login credentials in `.ha_login` (git-ignored).

- Reload YAML configs (automations, scripts, scenes, etc.)
- Restart Home Assistant
- Install/update HACS components
- Install new add-ons, restart add-ons
- Change integration settings
- Modify `configuration.yaml` and other config files
- View & control all entities
- Create/edit automations, scripts, helpers, scenes
- Build and deploy the React dashboard

## 🟡 Heads-Up to User

- Install new add-ons (mention what and why)

## 🔴 Never Without Explicit Approval

- Delete integrations, add-ons, or users
- Change network / WiFi / Tailscale settings
- Modify `secrets.yaml`
- Factory reset anything
- Change Olin's account settings
- Expose HA to the public internet
- Disable security features (SSL, auth, firewalls)
